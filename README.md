
# 🌾 AGRIVERSE 2.0
AgriVerse is an innovative agricultural management system designed to assist farmers in optimizing crop yield through data-driven insights and machine learning. The platform provides intelligent crop recommendations, yield prediction, and crop-specific information to help farmers make informed agricultural decisions.
##  Features
###  Crop Recommendation
Based on soil and climatic conditions, the system recommends the most suitable crop for cultivation. The recommendation model considers:
* Nitrogen (N)
* Phosphorous (P)
* Potassium (K)
* Soil pH
* Temperature
* Humidity
* Rainfall
###  Yield Prediction
Predicts the estimated crop yield based on:
* Crop Name
* State
* District
* Season
* Cultivation Area
The model incorporates geographical and climatic variations to provide accurate yield estimates.
###  Crop Information
Provides detailed information about various crops, including:
* Optimal Temperature
* Ideal Rainfall Range
* Suitable Humidity Conditions
* Recommended Growing Environment
###  Data Collection & Model Improvement
Verified agricultural data collected from farmers can be utilized to continuously improve the performance and accuracy of prediction models.

## ⚙️ Installation
### Clone Repository
```bash
git clone https://github.com/sathwik123677/AGRIVERSE.git
cd AGRIVERSE
```
### Backend Setup
```bash
cd Backend
pip install -r requirements.txt
python app_for_crop_recommendation.py
```
### Frontend Setup
```bash
cd FrontEnd
npm install
npm run dev
```
